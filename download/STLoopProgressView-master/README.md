## 类说明

源代码在TCircleView文件夹中，一共分为两个类：`STLoopProgressView`与`STLoopProgressView+BaseConfiguration`。

### `STLoopProgressView `

核心功能实现

### `STLoopProgressView+BaseConfiguration`

相关配置，如

-   三段颜色的配置
-   起始与结束角度配置
-   线宽
-   起始方向配置，顺逆时针
